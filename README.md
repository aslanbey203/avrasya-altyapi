# Avrasya Altyapı

Tek sayfalık, mobil uyumlu kanal açma ve altyapı hizmetleri sitesi.

## Önizleme

`index.html` dosyasını tarayıcıda açın veya klasörde basit bir statik sunucu çalıştırın:

```bash
npx serve .
```

## Yayınlama

Klasör GitHub'a yüklenip Vercel üzerinde statik site olarak yayımlanabilir. Alan adı daha sonra bağlanabilir.
